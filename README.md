# Plumbing Business Website (HTML/CSS Project)

## Overview

This project is a multi-page business website built using HTML and CSS. The site simulates a professional plumbing company website and demonstrates foundational web development skills including structured layout, navigation, and external styling.

This project was created as part of coursework and further refined for portfolio use.
